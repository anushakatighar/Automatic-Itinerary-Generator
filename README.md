# Automatic Itinerary Generator

A simple and interactive web application that automatically generates a day-by-day travel itinerary based on the destination and number of days selected. The project is fully client-side and runs using HTML, CSS, and JavaScript without any backend requirements.

## Features
- Select a destination from all Indian states and union territories.
- Choose the number of travel days.
- Automatically generates suggested activities for each day.
- Dynamic background images change based on destination.
- Clean UI with background video and glass-effect circular form.
- No backend needed — everything runs in the browser.

## Tech Stack
- **HTML5** — structure and UI pages  
- **CSS3** — styling, animations, glassmorphism  
- **JavaScript (Vanilla JS)** — logic, itinerary generation, query parameter handling
